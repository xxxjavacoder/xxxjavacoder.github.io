Навички
Що я використовую у роботі?
HTML 5
Саме він створює каркас вашого сайту aбо програми, а п'ята версія дозволить мені створювати більш SEO-оптимізовану структуру вашого продукту
CSS 3
Ця мова стилів дозволяє мені створювати абсолютно будь-який зовнішній вигляд вашого сайту або програми. Все обмежується лише вашою фантазією!
JavaScript
Ця мова програмування дозволяє оживити все, що завгодно: слайдери, вікна, підказки, вкладки, отримання даних від сервера та багато іншого
JQuery
Бібліотека Jquery дозволить прискорити розробку. Без необхідності інтегрувати в проект ми її не будемо, але звичка роботи з нею присутня
Sass / SCSS
Це метамова (мова для опису іншої мови), яка спрощує та прискорює написання CSS-коду. Його часто називають препроцесором CSS — це означає, що Sass має синтаксис, з якого генерує CSS-код, зрозумілий будь-якому браузеру.
Laravel
Це безкоштовний PHP-фреймворк із відкритим вихідним кодом, спеціально розроблений для створення складних сайтів та веб-додатків. Дозволяє спростити автентифікацію, маршрутизацію, сесії, кешування, архітектуру програми, роботу з базою даних.
Vue.js
Це прогресивний фреймворк для розробки інтерфейсів і односторінкових веб-додатків на мові JavaScript.
Git
Найпопулярніша система контролю версій проекта. Це система, яка дозволяє відразу декільком розробникам зберігати та відстежувати зміни у файлах вашого проекту.
